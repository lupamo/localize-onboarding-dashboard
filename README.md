# Localize Onboarding Dashboard

The localize onboardig dashboard is a tool designed to help new Wikipedia contributors from Afria especially in regions with very few active editors, easily get started and make meaningful contributions.

Many African language Wikipedia such as Swahili, Hausa, Yoruba, Amharic and Zulu are significantly under represented, despite being spoken by millions of people. New contributors often face major challenge when they land on generic onboarding pages that don't reflect their context, language or immediate opportunities to contribute.

This project soles that problem by acting as a guided entry point into Wikimedia ecosystem helping users quickly discover where they are needed most and how they can start contributing right away.
